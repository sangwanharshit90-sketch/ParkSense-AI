# 🚀 ParkSense AI

AI-powered Smart Bike Parking Detection and Booking System using YOLOv8 and Flask.

---

## 📌 Project Overview

ParkSense AI is an intelligent parking management system that uses **computer vision (YOLOv8)** to detect bikes in an image and automatically estimate parking slot availability.  

It provides an **interactive web interface** where users can:
- Upload parking images
- Detect occupied and available slots
- Select parking slots visually
- Book slots in real-time
- View booking and detection history

---

## 🧠 Features

✅ AI-based bike detection using YOLOv8  
✅ Dynamic parking slot estimation  
✅ Interactive slot selection (visual grid)  
✅ Real-time booking system  
✅ User authentication (Login/Register)  
✅ Booking history tracking  
✅ Detection history with images  
✅ Delete individual / all history (CRUD operations)  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Flask (Python)  
- **AI Model:** YOLOv8 (Ultralytics)  
- **Database:** SQLite  
- **Libraries:** OpenCV, Ultralytics
